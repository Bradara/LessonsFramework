## 🚀 Как работи?

### Навигация

```
Основна директория
├── 📁 Математика
│   ├── 📁 Алгебра
│   │   ├── 📄 Урок 1
│   │   └── 📄 Урок 2
│   └── 📁 Геометрия
│       └── 📄 Урок 1
├── 📁 Физика
│   └── 📄 Механика
└── 📄 Увод
```

**Стъпки:**
1. **Начало** → Виждате: Математика, Физика, Увод
2. Кликнете **Математика** → Виждате: Алгебра, Геометрия
3. Кликнете **Алгебра** → Виждате: Урок 1, Урок 2
4. **Breadcrumb:** Начало > Математика > Алгебра

### Breadcrumb навигация

```
🏠 Начало > Математика > Алгебра
```

- Кликнете на **🏠 Начало** → Връщане в главното меню
- Кликнете на **Математика** → Връщане към папката Математика
- **Алгебра** е винаги текущата локация (не е кликаема)

## 🎨 Фонове на папките

### Как работят?

Всяка папка автоматично получава **красив CSS градиент**:

**Предимства:**
- ✅ Винаги работят (не зависят от интернет)
- ✅ Моментално зареждане
- ✅ 0 байта дисково пространство
- ✅ 12 различни цветови комбинации
- ✅ Консистентност - същата папка винаги има същия цвят

**Налични градиенти:**
1. 🔵 Сине-лилаво
2. 🌸 Розово-червено
3. 💙 Синьо
4. 💚 Зелено
5. 🌈 Розово-жълто
6. 🌊 Тюркоаз
7. 🎀 Пастелно
8. 💗 Нежно розово
9. 🍊 Портокалово
10. ❤️ Червено-синьо
11. 💜 Лилаво-синьо
12. 🔥 Огнено

**Логиката:**
- Името на папката се "хешира"
- Получава се индекс от 0 до 11
- Същата папка винаги има същия градиент
- Различни папки имат различни градиенти

### Примери

| Папка | Градиент | Цветове |
|-------|----------|----------|
| Математика | Синьо | Светло → Тъмно синьо |
| Физика | Зелено | Ярко зелено → Тюркоаз |
| Програмиране | Лилаво | Фиолетово → Розово |
| История | Портокалово | Кремаво → Корал |

## 📝 Създаване на йерархична структура

### Метод 1: Ръчно

```powershell
# Създайте главна папка
mkdir "Математика"

# Създайте подпапки
mkdir "Математика\Алгебра"
mkdir "Математика\Геометрия"

# Добавете уроци
copy lesson_template.html "Математика\Алгебра\Урок 1.html"
copy lesson_template.html "Математика\Геометрия\Урок 1.html"

# Сканирайте рекурсивно
python scanner.py -r
```

### Метод 2: Партидно създаване

```powershell
# Създайте цяла структура наведнъж
$structure = @(
    "Математика\Алгебра",
    "Математика\Геометрия",
    "Математика\Анализ",
    "Физика\Механика",
    "Физика\Термодинамика",
    "Химия\Органична",
    "Химия\Неорганична"
)

foreach ($path in $structure) {
    New-Item -ItemType Directory -Force -Path $path
}

# Сканирайте
python scanner.py -r
```

## 🔧 Персонализация

### Промяна на темите за фонове

Редактирайте `index.html`, намерете:

```javascript
const backgroundTopics = [
    'education', 'learning', 'books', 'study', 'classroom', 
    'science', 'mathematics', 'technology', 'nature', 'abstract'
];
```

Можете да добавите свои теми:
```javascript
const backgroundTopics = [
    'space',        // космос
    'ocean',        // океан
    'mountains',    // планини
    'art',          // изкуство
    'music',        // музика
    'architecture', // архитектура
    'city',         // град
    'forest'        // гора
];
```

### Фиксиран фон за конкретна папка

Ако искате определена папка да има винаги същия фон:

```javascript
const getFolderBackground = (folderName) => {
    // Специални фонове за конкретни папки
    const customBackgrounds = {
        'Математика': 'https://source.unsplash.com/1600x900/?mathematics',
        'Физика': 'https://source.unsplash.com/1600x900/?physics',
        'Химия': 'https://source.unsplash.com/1600x900/?chemistry'
    };
    
    if (customBackgrounds[folderName]) {
        return customBackgrounds[folderName];
    }
    
    // За останалите - автоматична тема
    const hash = folderName.split('').reduce((acc, char) => {
        return char.charCodeAt(0) + ((acc << 5) - acc);
    }, 0);
    const index = Math.abs(hash) % backgroundTopics.length;
    const topic = backgroundTopics[index];
    return `https://source.unsplash.com/1600x900/?${topic}`;
};
```

### Без фонове (само цвят)

Ако предпочитате само градиенти вместо изображения:

В `Card` компонента, променете:

```javascript
{isFolder && (
    <div 
        className="h-32 bg-gradient-to-br from-purple-400 to-pink-500"
    >
    </div>
)}
```

## 💡 Съвети

### За добра организация

✅ **DO:**
- Използвайте ясни, описателни имена на папките
- Групирайте свързани уроци в една папка
- Ограничете дълбочината до 2-3 нива
- Правете логическа йерархия (Тема > Раздел > Урок)

❌ **DON'T:**
- Не създавайте твърде дълбоки йерархии (4+ нива)
- Не използвайте твърде дълги имена на папки
- Не забравяйте `python scanner.py -r` след промени

### Оптимизация на скоростта

Фоновете се зареждат от Unsplash, което може да е бавно. За по-бързо зареждане:

1. **Локални изображения:**
   - Създайте папка `backgrounds/`
   - Добавете свои изображения
   - Променете пътя в `getFolderBackground()`

2. **Cache:**
   - Изображенията се кешират от браузъра
   - При повторно посещение са моментални

## 🐛 Troubleshooting

### Не виждам фона на папката

**Не е възможно!** Градиентите винаги работят. Ако все пак не виждате фон:

**Проверете:**
1. Browser cache - натиснете Ctrl+F5
2. JavaScript errors - F12 → Console
3. Уверете се, че сте на `http://localhost:8000` (не `file://`)

### Искам изображения вместо градиенти

**Решение:** Вижте секцията "Персонализация" по-горе или [backgrounds/README.md](backgrounds/README.md)

### Папката е празна, но има файлове

**Причина:** Файловете са твърде дълбоко вложени или scanner не ги е открил

**Решение:**
```powershell
# Проверете structure.json
cat structure.json

# Регенерирайте с -r флаг
python scanner.py -r

# Проверете имената на файловете (.html ли са?)
ls -Recurse -Filter *.html
```

### Breadcrumb не работи

**Причина:** JavaScript грешка или остаряла версия на браузъра

**Решение:**
- Натиснете F12 → Console → проверете за грешки
- Опитайте в друг браузър (Chrome, Firefox, Edge)
- Hard refresh: Ctrl+F5
