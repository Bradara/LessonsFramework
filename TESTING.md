## Стъпка 4: Добавете собствено съдържание ✏️

### Метод 1: Ръчно създаване

```powershell
# Създайте нова папка
mkdir "Моите уроци"

# Копирайте шаблона
copy lesson_template.html "Моите уроци\Първи урок.html"

# Регенерирайте структурата
python scanner.py -r
```

### Метод 2: Използване на template

1. Отворете `lesson_template.html` във VS Code
2. Редактирайте съдържанието
3. Save As → "Моите уроци\Нов урок.html"
4. Стартирайте `python scanner.py -r`
5. Refresh браузъра (F5)

## Стъпка 5: Тестване на scanner.py 🔍

```powershell
# Помощ
python scanner.py --help

# Само текущо ниво
python scanner.py

# Рекурсивно (препоръчително)
python scanner.py -r

# С различен изход
python scanner.py -r -o my-structure.json
```

## Типични проблеми и решения 🔧

### Проблем 1: "Не може да се зареди structure.json"

**Решение:**
```powershell
python scanner.py -r
```

### Проблем 2: Не виждам новите уроци

**Решение:**
1. Уверете се, че файлът има `.html` разширение
2. Стартирайте `python scanner.py -r`
3. Refresh браузъра (Ctrl+F5)

### Проблем 3: Python грешка

**Решение:**
```powershell
# Проверете версията
python --version

# Трябва да е 3.6 или по-нова
```

### Проблем 4: "NetworkError when attempting to fetch resource"

**Решение:**
```powershell
# Стартирайте HTTP сървър вместо директно отваряне
python -m http.server 8000
# Отворете: http://localhost:8000
```

### Проблем 5: Кирилица не се показва правилно

**Решение:**
- Уверете се, че файлът е запазен с UTF-8 encoding във VS Code
- Проверете дали имате `<meta charset="UTF-8">` в HTML
