Основите на HyperText Markup Language

Какво е HTML?

HTML (HyperText Markup Language) е стандартният език за създаване на уеб страници. Това е markup език, който казва на браузъра как да структурира съдържанието на страницата.

Важно: HTML не е програмен език - той е markup език, който използва тагове за структуриране на съдържание.

Основна структура

Всяка HTML страница има основна структура, която включва: 

<!DOCTYPE html>
<html lang="bg">
<head>
    <meta charset="UTF-8">
    <title>Заглавие на страницата</title>
</head>
<body>
    <h1>Здравей, свят!</h1>
    <p>Това е моята първа HTML страница.</p>
</body>
</html>

Обяснение на елементите

  • <!DOCTYPE html> - декларира, че документът е HTML5
  • <html> - коренният елемент на страницата
  • <head> - съдържа мета информация
  • <body> - съдържа видимото съдържание

Основни тагове

Заглавия

HTML предлага 6 нива на заглавия от <h1> до <h6>:

<h1>Най-важно заглавие</h1>
<h2>Подзаглавие</h2>
<h3>По-малко важно</h3>

Параграфи и текст

<p>Това е параграф.</p>
<strong>Удебелен текст</strong>
<em>Курсив текст</em>
<br> <!-- Нов ред -->

Списъци

Неподреден списък (с bullets):

<ul>
    <li>Първи елемент</li>
    <li>Втори елемент</li>
</ul>

Подреден списък (с номера):

<ol>
    <li>Първа стъпка</li>
    <li>Втора стъпка</li>
</ol>

Връзки и изображения

<!-- Връзка -->
<a href="https://example.com">Кликни тук</a>

<!-- Изображение -->
<img src="image.jpg" alt="Описание">

Семантични тагове

HTML5 въведе семантични тагове, които правят кода по-четим:

  • <header> - горна част на страницата
  • <nav> - навигационно меню
  • <main> - основно съдържание
  • <article> - статия или самостоятелно съдържание
  • <section> - секция от съдържание
  • <footer> - долна част на страницата

Предизвикателство: Опитайте да създадете проста HTML страница с всички елементи, които научихте! Използвайте заглавия, параграфи, списък и връзка.

Заключение

HTML е основата на всяка уеб страница. След като овладеете основите, можете да продължите с CSS за стилизиране и JavaScript за интерактивност. Практиката е ключова - колкото повече експериментирате, толкова по-добре ще разберете!

Следващи стъпки

  1. Създайте собствена HTML страница
  2. Експериментирайте с различни тагове
  3. Научете за HTML атрибути
  4. Запознайте се с CSS за стилизиране
