# 🎬 НОВОСТИ! Навигация между папки + Уникални фонове

## ✨ Какво е ново в последната версия?

### 1. 📁 Навигация между папки

**Преди:**
- Всички папки и уроци се показваха заедно на едно ниво
- Трудно за навигация при много съдържание

**Сега:**
- ✅ Кликнете на папка → влизате в нея
- ✅ Виждате само съдържанието на текущата папка
- ✅ Неограничена дълбочина (папки в папки в папки...)
- ✅ Breadcrumb навигация в горната част

**Пример:**
```
🏠 Начало → Виждате: Математика, Физика, Програмиране
   ↓ (кликнете Математика)
🏠 Начало > Математика → Виждате: Алгебра, Геометрия
   ↓ (кликнете Алгебра)
🏠 Начало > Математика > Алгебра → Виждате: Урок 1, Урок 2
```

### 2. 🎨 Уникални градиенти за папки

**Нова визия:**
- Всяка папка автоматично получава красив header с градиент
- 12 различни цветови комбинации
- Моментално зареждане (без зависимост от интернет)
- Консистентност - същата папка винаги има същия цвят
- Опционално: Поддръжка на локални изображения

**Предимства:**
- ✅ Винаги работят
- ✅ Бързи (0ms зареждане)
- ✅ Не заемат място
- ✅ Красиви и модерни

**Как изглежда:**
```
┌──────────────────────────────┐
│  [Красиво фоново изображение]│
│            📁                │
│      Математика              │
│         Папка                │
└──────────────────────────────┘
```

### 3. 🧭 Breadcrumb навигация

В горната част на всяка страница:
```
🏠 Начало > Математика > Алгебра
```

- Кликнете на **🏠 Начало** → Върви в главното меню
- Кликнете на **Математика** → Връща ви в папката Математика
- **Алгебра** е текущата локация (не може да се кликне)

## 🚀 Как да използвате новите функции?

### Създаване на йерархична структура

```powershell
# Създайте главна папка
mkdir "Математика"

# Създайте подпапки
mkdir "Математика\Алгебра"
mkdir "Математика\Геометрия"
mkdir "Математика\Анализ"

# Добавете уроци в подпапките
copy lesson_template.html "Математика\Алгебра\Урок 1.html"
copy lesson_template.html "Математика\Алгебра\Урок 2.html"
copy lesson_template.html "Математика\Геометрия\Урок 1.html"

# Сканирайте РЕКУРСИВНО (важно!)
python scanner.py -r

# Отворете в браузър
# Ще видите само "Математика" във главното меню
# Кликнете върху нея → ще видите Алгебра, Геометрия, Анализ
# Кликнете на Алгебра → ще видите Урок 1 и Урок 2
```

### Пример за сложна структура

```powershell
# Създайте цяла образователна система
mkdir "10 клас\Математика"

mkdir "11 клас\Математика"

# Сканирайте

# Навигация:
# Начало → 10 клас → Математика → Урок 1
#       ↓ 11 клас → Физика → Механика
#       ↓ 12 клас → ...
```

## 💡 Съвети и трикове

### 1. Използвайте значими имена

❌ Лошо:
```
Папка1/
  Файл1.html
  Файл2.html
```

✅ Добре:
```
Математика/
  Алгебра/
    Линейни уравнения.html
    Квадратни уравнения.html
  Геометрия/
    Триъгълници.html
```

### 2. Ограничете дълбочината

✅ Препоръчително (2-3 нива):
```
Програмиране → Python → Основи → Променливи
```

❌ Избягвайте (4+ нива):
```
Образование → Училище → 10 клас → Математика → Алгебра → Уравнения → Линейни
```

### 3. Тествайте редовно

След всяка промяна:
```powershell
python scanner.py -r   # Регенерирайте структурата
# Ctrl+F5 в браузъра   # Hard refresh
```

## 🎨 Персонализация на фоновете

### Промяна на темите

Редактирайте [index.html](index.html), намерете:

```javascript
const backgroundTopics = [
    'education', 'learning', 'books', ...
];
```

Променете на свои:
```javascript
const backgroundTopics = [
    'space',      // космос
    'ocean',      // океан  
    'mountains',  // планини
    'forest'      // гора
];
```

### Фиксиран фон за конкретна папка

Ако искате "Математика" да има винаги един и същ фон:

```javascript
const getFolderBackground = (folderName) => {
    // Специални фонове
    if (folderName === 'Математика') {
        return 'https://source.unsplash.com/1600x900/?mathematics';
    }
    if (folderName === 'Физика') {
        return 'https://source.unsplash.com/1600x900/?physics';
    }
    
    // За останалите - автоматично
    // ... (оригинален код)
};
```

### Локални фонове (по-бързо)

Ако Unsplash е бавен:

1. Създайте папка `backgrounds/`
2. Добавете изображения: `math.jpg`, `physics.jpg`, etc.
3. Променете функцията:

```javascript
const getFolderBackground = (folderName) => {
    const backgrounds = {
        'Математика': 'backgrounds/math.jpg',
        'Физика': 'backgrounds/physics.jpg'
    };
    return backgrounds[folderName] || 'backgrounds/default.jpg';
};
```

## 📊 Преди vs. Сега

### Преди (плоска структура)
```

- 📁 Математика
- 📁 Физика
- 📁 Химия
- 📁 Алгебра
- 📁 Геометрия
- 📄 Урок 1
- 📄 Урок 2
- 📄 Урок 3
... (всичко наведнъж - объркващо!)
```

```
Dashboard (Начало):
- 📁 Математика
- 📁 Физика
- 📁 Химия

→ Кликнете Математика:
- 📁 Алгебра
- 📁 Геометрия

→ Кликнете Алгебра:
- 📄 Урок 1
- 📄 Урок 2
```

**Резултат:** Чисто, организирано, лесно за навигация! ✨

## 🆘 Trouble shooting

### Не се зареждат фоновете

**Причина:** Unsplash API може да е блокиран

**Решение:**
1. Проверете интернет връзката
2. Използвайте локални изображения (вижте по-горе)
3. Или премахнете фоновете (използвайте само градиенти)

### Папките не се показват правилно

**Причина:** Не сте използвали `-r` флаг

**Решение:**
```powershell
python scanner.py -r  # ВАЖНО: -r за рекурсия!
```

### Breadcrumb не работи

**Причина:** Browser cache

**Решение:**
```
Ctrl + F5  (Windows)
Cmd + Shift + R  (Mac)
```
