# 🚀 Бърз старт за начинаещи

## ✅ Как да стартирам портала?

### ⚠️ ВАЖНО: НЕ отваряйте `index.html` директно!

Ако отворите `index.html` с двоен клик, ще получите грешка:
```
❌ NetworkError when attempting to fetch resource
```

### ✅ ПРАВИЛЕН начин:

#### Вариант 1: Python HTTP Server (Най-лесен)

1. Отворете PowerShell или CMD в папката на проекта
2. Стартирайте:
```powershell
python -m http.server 8000
```
3. Отворете браузър на адреса:
```
http://localhost:8000
```

**Готово!** Трябва да видите Dashboard-а с вашите уроци. 🎉

#### Вариант 2: VS Code Live Server (Най-добър за разработка)

1. Отворете `index.html` във VS Code
2. Щракнете с **десен бутон** на файла
3. Изберете **"Open with Live Server"**
4. Браузърът ще се отвори автоматично

**Предимство:** Всяка промяна автоматично се обновява в браузъра!

## 🔥 Пълен workflow за нов урок

```powershell
# 1. Създайте папка и подпапка (по желание)
mkdir "Математика\Алгебра"

# 2. Копирайте шаблона
copy lesson_template.html "Математика\Алгебра\Урок 1.html"

# 3. Редактирайте урока
code "Математика\Алгебра\Урок 1.html"

# 4. Генерирайте структурата (ВАЖНО: -r за рекурсия!)
python scanner.py -r

# 5. Стартирайте сървъра (ако не работи вече)
python -m http.server 8000

# 6. Отворете http://localhost:8000
# 7. Кликнете на папка "Математика" → "Алгебра" → Виждате урока!
```

## ❓ Често задавани въпроси

### Защо не мога просто да отворя index.html?

**Кратък отговор:** Поради сигурност, браузърите блокират зареждането на локални файлове чрез JavaScript.

**Техничен отговор:** Когато отваряте файл като `file:///C:/...`, браузърът прилага CORS (Cross-Origin Resource Sharing) политики, които блокират `fetch()` заявките към други локални файлове (`structure.json`). HTTP сървърът решава този проблем, защото обслужва файловете през `http://` протокол.

### Как да спра сървъра?

В терминала, където работи `python -m http.server`, натиснете:
```
Ctrl + C
```

### Виждам "Address already in use"?

Порт 8000 вече се използва. Използвайте друг порт:
```powershell
python -m http.server 8001
# Отворете http://localhost:8001
```

### Как да добавя нов урок без да създавам папка?

```powershell
# Копирайте шаблона в главната папка
copy lesson_template.html "Нов урок.html"

# Редактирайте
code "Нов урок.html"

# Сканирайте
python scanner.py

# Refresh браузъра
```

## 📱 Тествайте на телефон

Ако сте на същата WiFi мрежа:

1. Намерете вашия IP адрес:
```powershell
ipconfig
# Търсете IPv4 Address (напр. 192.168.1.100)
```

2. Стартирайте сървъра:
```powershell
python -m http.server 8000
```

3. На телефона отворете:
```
http://192.168.1.100:8000
```

## 🆘 Помощ

### Виждам празна страница

1. Проверете дали `structure.json` съществува:
```powershell
ls structure.json
```

2. Ако не съществува, създайте го:
```powershell
python scanner.py -r
```

3. Refresh браузъра (F5)

### Не виждам новия урок

1. Регенерирайте структурата:
```powershell
python scanner.py -r
```

2. Hard refresh в браузъра:
```
Ctrl + F5 (Windows/Linux)
Cmd + Shift + R (Mac)
```

### Python грешка

Проверете дали имате Python 3.6+:
```powershell
python --version
```

Ако нямате Python, свалете от: https://python.org
